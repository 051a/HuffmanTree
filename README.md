# 哈夫曼树（Huffman Tree）应用程序

> 数据结构与算法课程设计

## 项目简介

基于哈夫曼编码算法，实现对 27 个字符（空格 + 26 个大写英文字母）的编码与解码。提供 C 语言和 C++ 两种实现。

## 功能

| 功能 | 说明 |
|------|------|
| **构建哈夫曼树** | 根据给定字符频率，自底向上构建哈夫曼树 |
| **生成编码表** | 从叶子节点回溯到根，生成每个字符的哈夫曼编码 |
| **打印树结构** | 横向缩进显示哈夫曼树的完整结构 |
| **计算传输长度** | 计算字符串 "THIS PROGRAM IS MY FAVORITE" 的报文总传输位数 |
| **解码 01 串** | 根据哈夫曼树对用户输入的 01 编码串进行解码 |

## 构建规则

- 左孩子权值 ≤ 右孩子权值
- 权值相同时，原序列中先出现的作为左孩子

## 编译运行

### Visual Studio

1. 打开 `HuffmanTree.sln`
2. 项目属性 → C/C++ → 命令行 → 其他选项 → 添加 `/utf-8`（C 版本需要）
3. 编译运行

### 命令行（C++）

```bash
g++ HuffmanTree.cpp -o HuffmanTree.exe -fexec-charset=utf-8
./HuffmanTree.exe
```

### 命令行（C）

```bash
gcc HuffmanTree.c -o HuffmanTree.exe -finput-charset=utf-8 -fexec-charset=utf-8
./HuffmanTree.exe
```

## 数据结构

```c
typedef struct {
    char  ch;         // 字符（叶子结点有效）
    int   weight;     // 权值
    int   parent;     // 双亲下标，-1 表示无
    int   lchild;     // 左孩子下标，-1 表示无
    int   rchild;     // 右孩子下标，-1 表示无
    int   order;      // 在原序列中的出现次序
} HTNode;

typedef struct {
    char ch;          // 字符
    char bits[200];   // 编码 01 串
    int  len;         // 编码长度
} HTCode;
```
